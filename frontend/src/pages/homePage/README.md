# 首页模块 (HomePage)

## 模块说明
首页模块主要提供智能 AI 对话系统的核心界面和交互逻辑，包括会话管理、多模态消息发送、流式响应处理等功能。

## 核心功能
1. **多模型支持**：支持切换不同的大语言模型（如 Ollama、OpenAI 等）。
2. **会话管理**：创建、切换和删除对话会话。
3. **多模态输入**：支持发送文本和图片（最多 9 张图片，单张 5MB 以内）。
4. **流式输出与打字机特效**：
   - 自动解析流式数据协议（如 SSE、fetch-readable-stream），将 chunk 顺序追加。
   - 提供自动纯化 JSON 内容能力，仅显示 Markdown 文本。
   - AI 消息返回时，实现平滑的“打字机”动画（约 30-60ms/字符）。
   - 支持动态中断：在组件卸载或新消息发送时，通过 `AbortController` 立即中断旧流并清理定时器，有效防止内存泄漏。
5. **动态卡片渲染**：支持解析 `[[card:type|payload]]` 和 `{{card:id}}` 协议渲染自定义 Vue 组件。
6. **思维过程展示**：解析 `<think>` 标签并展示 AI 的推理过程。

## 测试覆盖
本模块的 `MessageContent.vue` 组件包含完善的测试：
- **单元测试 (`vitest` + `vue-test-utils`)**：覆盖了打字机特效的各种场景（正常输出、中断流、暂停/继续、清理定时器避免内存泄漏）及 JSON 内容提取纯化。
- **E2E 测试 (`cypress`)**：在 `cypress/e2e/chat-stream.cy.js` 中模拟了后端的真实 SSE 流响应，涵盖了正常流、中断流以及错误流三种核心交互场景。
